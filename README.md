# Fiona's birthday invitation
An interactive storybook built with p5.js

## Setup
```
git clone [tbd]
npm install
```

## Deploy
This project is deployed on [surge.sh](https://surge.sh/) at [tbd](http://tbd.surge.sh/)

```
npm run deploy
```
